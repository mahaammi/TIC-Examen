Chapitre 4

  • LE WEB ET L’INTERNET :

    • Le World Wide Web (WWW) est un système hypertexte public sur Internet qui permet de parcourir des pages accessibles sur des sites. Internet est un support de communication général, tandis que le Web n'est qu'une partie des contenus circulant sur ce dernier.

  • ASPECT TECHNIQUE DU WEB :

    • Les trois mécanismes de base du Web sont l'URL (Uniform Resource Locator), le protocole HTTP (HyperText Transfer Protocol) et le langage de description de pages Web, HTML (HyperText Markup Langage)

  • PAGE WEB ET SITE WEB :

    • Une page web est un document multimédia représenté par divers éléments à l'aide d'un langage de description sauvegardé dans un fichier, et cette dernière est visualisée par un navigateur et localisée grâce à une URL. Tandis qu'un site web est un rassemblement de page web de même sujet jointes entre elles par des liens hypertextes et dérivées d'une entité et d'une URL commune.

  • CREATION D’UNE PAGE WEB HTML :

    • Pour réaliser une page Web, il faut d'abord créer un document HTML en utilisant un éditeur de pages Web ou éditeur de texte et respecter les principes du langage HTML en se basant sur des balises telles que 'html' (début du document), 'Head' (entête du document), 'title' (titre du document), et 'body' (corps du document), ainsi que des titres de niveau et des paragraphes, et ce document sera visualisé dans un navigateur quelconque.
