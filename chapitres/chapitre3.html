Chapitre 3

    1. Les Composants matériel d’un Réseau informatique

      • Les cartes réseau :

        Sont également appelées carte d’interface réseau (NIC), sont des composants matériels qui permettent à un ordinateur de se connecter à un réseau, sont souvent intégrées à l’intérieur de l’ordinateur, sont utilisée pour se connecter à différent type de réseaux tels que les réseaux locaux (LAN), les réseaux étendus (WAN) et internet. Les cartes réseau permettent également de transférer et de recevoir des données entre différent ordinateur. Il existe différent type de carte réseau y compris les cartes Ethernet, Wi -fi, modem, etc.

      • Switch :

        Un switch est un équipement qui permet de connecter et de transférer des données entre des périphériques dans un réseau local. Il utilise des adresses MAC pour identifier les périphériques et envoyer les données au bon destinataire. Il existe différents types de switch selon les normes de communication qu’ils supportent.

      • Hub :

        Un hub est un équipement qui permet de connecter des périphériques dans un réseau local, mais qui envoie les données à tous les périphériques connectés, ce qui ralentit et réduit la sécurité du réseau. Les hubs sont moins performants que les switches et sont utilisés pour des tâches simples dans des petits réseaux.

    2. Le Modèle OSI :

      3. OSI (Open Systems Interconnection) est un modèle de référence pour les réseaux informatique il est composé de sept couches, chacune représentant une fonction différente dans la communication de réseau . Les sept couches sont :

      • La couche physique .
      • La couche liaison de donnée.
      • La couche réseau.
      • La couche transport.
      • La couche session.
      • La couche présentation.
      • La couche application.

        • Le modèle OSI est un cadre pour comprendre la communication de réseau, il n’est pas utilisé dans les protocoles actuels comme TCP/IP, qui ont un autre modèle de référence.

    3. Le Modèle TCP/IP(Transmission Contrôle Protocole/Internet Protocole):

      4. est un modèle de référence pour les communications de réseau, ce dernier est utilisé pour la conception des réseaux informatiques modernes, il divise les communications en quatre couches principales :

      • La couche liaison de données.
      • La couche réseau.
      • La couche transport.
      • La couche application.

        • Le modèle TCP/IP est le modèle de référence pour les protocoles de communication de réseau les plus largement utilisés, Il est utilisé pour là Conception et la mise en œuvre des réseaux.

    4. L’adressage:

      5. est une règle qui permet d'identifier de manière unique les deux unités en communication dans un réseau. 'Un adressage IP' :Une adresse IP (Internet Protocole) est le numéro qui identifie chaque ordinateur dans les réseaux qu’il appartient. Il y a deux modes d’adressage : IPV4 et IPV6, celles-ci diffèrent en termes de nombre de bits utilisés pour représenter l'adresse.

    5. L’adresse IPV4:

      6. elle est divisée en deux parties composées de 2 octets : la partie réseau qui contient les bits de valeur supérieure et correspond à l'adresse réseau, et la partie hôte qui contient les bits de valeur inférieure et correspond à l'adresse hôte. On distingue les deux parties comme suit : hôte si la valeur des bits pour la partie réseau est la même. Le nombre de machines possibles sur ce réseau est déterminé par le nombre de bits contenus dans la partie hôte. L’adressage IPV4 s’effectué selon les types suivants :

      • L’adresse réseau.
      • L’adresse de diffusion.
      • L’adresse hôte
      • Le masque de réseau et de sous réseaux.
